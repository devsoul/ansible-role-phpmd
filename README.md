PHP Mess Detector Ansible role
=========

If you are a PHP developer and you use PHP Mess Detector.

Requirements
------------
  - minimum `php 7` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phpmd

Check if it's installed:
------
